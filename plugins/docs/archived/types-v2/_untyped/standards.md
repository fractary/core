# Untyped Documentation Standards

## Required Conventions

### 1. Basic Structure
- ALWAYS include title and description
- ALWAYS use proper markdown formatting
- ALWAYS include author and dates
- ALWAYS maintain version number

### 2. Content Quality
- ALWAYS write clear, concise content
- ALWAYS use proper grammar and spelling
- ALWAYS structure content with headings
- ALWAYS keep content up to date

## Best Practices

- Consider using a specific doc_type if applicable
- Follow general documentation best practices
- Keep formatting consistent
- Include relevant tags for discoverability
