# Doc Delete

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | id |
| `--json` | No | json |
| `--context` | No | context |

## Execution

Delete a document using the CLI command `fractary-core docs doc-delete`.

Examples:
- `fractary-core docs doc-delete guide-001`
- `fractary-core docs doc-delete adr-001 --json`
