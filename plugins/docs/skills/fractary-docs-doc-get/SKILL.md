# Doc Get

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | id |
| `--json` | No | json |
| `--context` | No | context |

## Execution

Get a document using the CLI command `fractary-core docs doc-get`.

Examples:
- `fractary-core docs doc-get guide-001`
- `fractary-core docs doc-get adr-001 --json`
