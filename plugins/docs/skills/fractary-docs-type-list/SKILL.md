# Type List

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `--json` | No | json |
| `--context` | No | context |

## Execution

List available document types using the CLI command `fractary-core docs type-list`.

Examples:
- `fractary-core docs type-list`
- `fractary-core docs type-list --json`
