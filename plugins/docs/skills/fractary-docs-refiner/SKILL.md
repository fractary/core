# Docs Refiner

Scans documents for gaps, generates prioritized questions, and applies improvements through interactive Q&A. Works for any document type.

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | Document ID to refine |
| `--context "<text>"` | No | Focus areas or additional instructions |

## Execution

Read `docs/refine-flow.md` and follow the refinement workflow.
