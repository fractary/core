# Debug Log Standards

## Purpose
Capture debugging sessions for issue investigation and problem-solving.

## Required Sections
- Problem Description
- Steps to Reproduce
- Debug Output
- Analysis
- Solution (if resolved)

## Capture Rules
**ALWAYS capture**: Stack traces, error messages, variable values, system state
**REDACT**: Production data, credentials, PII

## Retention
- Local: 7 days
- Cloud: 30 days
- Priority: medium
