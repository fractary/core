# Types

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `--json` | No | json |
| `--context` | No | context |

## Execution

List available log types using the CLI command `fractary-core logs types`.

Examples:
- `fractary-core logs types`
- `fractary-core logs types --json`
