# Stop

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `--json` | No | json |
| `--context` | No | context |

## Execution

Stop the active session capture using the CLI command `fractary-core logs stop`.

Examples:
- `fractary-core logs stop`
- `fractary-core logs stop --json`
